# Apple Notes Skill
Use the `memo` CLI to manage Apple Notes.
## Commands
- `memo list` - List all notes
- `memo show <id>` - View a note
- `memo create --title "Title" --body "Content"` - Create note
- `memo search "query"` - Search notes
