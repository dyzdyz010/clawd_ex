# GitHub Skill
Use `gh` CLI for GitHub interactions.
## Common Commands
- `gh issue list` - List issues
- `gh pr list` - List PRs
- `gh pr create` - Create PR
- `gh run list` - List CI runs
- `gh api` - Advanced API queries
